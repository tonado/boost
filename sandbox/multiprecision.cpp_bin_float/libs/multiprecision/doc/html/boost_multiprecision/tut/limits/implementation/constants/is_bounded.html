is_bounded

Constant std::numeric_limits<T>::is_bounded is true if there is a limit to values that can be stored.

This is obviously true for all built-in integer, fixed and floating-point types, but will be false for arbitrary precision integer (for example cpp_int) and floating-point types because these expand until memory is exhausted.
