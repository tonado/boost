deque<T,NodeGenerator,Balancer>

Description

The deque data structure is a double-ended queue whose programming interface is very similar to that of boost::container::deque but whose performance characteristics are significantly different: depending on the Balancer type in use, it can be expanded or contracted at both ends in amortized logarithmic time with respect to the number of elements, and any element can be accessed by index in logarithmic time as well. Though this worst-case performance is not as good as the amortized constant time guaranteed by the corresponding boost::container::deque operations, the trade-off may be acceptable because this data structure also allows elements to be inserted to or removed from arbitrary positions in amortized logarithmic time, which is much better than the linear time of the corresponding boost::container::deque operations.
