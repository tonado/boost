The BOOST_PP_ASSERT_IS_ARRAY macro conditionally causes a preprocessing error if its parameter is not an array.
Although the macro itself is not a variadic macro, it needs variadic macro support in order to be invoked.

Usage

BOOST_PP_ASSERT_IS_ARRAY(param) (v)

Arguments

param
A parameter to check for an array.

Remarks

If param is not an array, this macro causes a preprocessing error.  Otherwise, it expands to nothing.

See Also

  • BOOST_PP_ASSERT_IS_LIST
  • BOOST_PP_ASSERT_IS_SEQ
  • BOOST_PP_ASSERT_IS_TUPLE

Requirements

Header:  <boost/preprocessor/debug/assert_is_array.hpp>

Sample Code

#include <boost/preprocessor/debug/assert_is_array.hpp>

#define AN_ARRAY (3,(a,b,c))
#define NOT_JUST_ARRAY (3,(a,b,c)) text
#define A_TUPLE (a,b,c)
#define A_LIST (a,(b,(c,BOOST_PP_NIL)))
#define A_SEQ (a)(b)(c)
#define A_TEXT text

BOOST_PP_ASSERT_IS_ARRAY(AN_ARRAY) // no output
BOOST_PP_ASSERT_IS_ARRAY(NOT_JUST_ARRAY) // generates preprocessing error
BOOST_PP_ASSERT_IS_ARRAY(A_TUPLE) // generates preprocessing error
BOOST_PP_ASSERT_IS_ARRAY(A_LIST) // generates preprocessing error
BOOST_PP_ASSERT_IS_ARRAY(A_SEQ) // generates preprocessing error
BOOST_PP_ASSERT_IS_ARRAY(A_TEXT) // generates preprocessing error
