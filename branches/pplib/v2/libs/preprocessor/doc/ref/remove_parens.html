The BOOST_PP_REMOVE_PARENS macro expands to its argument with the starting pair of parens removed if it exists. Although the macro itself is not a variadic macro, it needs variadic macro support in order to be invoked.

Usage

BOOST_PP_REMOVE_PARENS(param) (v)

Arguments

param
The parameter whose parens will be removed.

Remarks

This macro returns its argument with the starting pair of parens removed. If param does not start with parens, the macro just returns its argument.
Requirements
Header:  <boost/preprocessor/facilities/remove_parens.hpp>

Sample Code

#include <boost/preprocessor/facilities/remove_parens.hpp>

#define ONLY_PARENS (a,b,c)
#define STARTS_WITH_PARENS (a,b,c) cheers (d,e)
#define ENDS_WITH_PARENS cheers (a,b,c)

BOOST_PP_REMOVE_PARENS(ONLY_PARENS) // output is: a,b,c
BOOST_PP_REMOVE_PARENS(STARTS_WITH_PARENS) // output is: a,b,c cheers (d,e)
BOOST_PP_REMOVE_PARENS(ENDS_WITH_PARENS) // output is: cheers (a,b,c)
